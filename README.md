# Employee Task Manager

A role-based Employee Task Management System built using Flask and SQLite.

## Features

- Admin authentication
- Employee authentication
- Role-based dashboards
- Add employees
- Assign tasks to employees
- Employees update task status
- Relational database with foreign keys
- Error handling for duplicate users

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- Jinja2

## How to Run

```bash
git clone https://github.com/anishkb-dev/employee-task-manager.git
cd employee-task-manager
python3 -m venv venv
source venv/bin/activate
pip install flask
python app.py
